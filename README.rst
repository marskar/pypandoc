========================
pypandoc
========================


A binder-enabled repo for making HTML slides using pypandoc.


Project structure
=================

::

    pypandoc
    ├── AUTHORS.rst
    ├── CHANGELOG.rst
    ├── LICENSE.txt
    ├── Makefile
    ├── README.rst
    ├── docs
    │   ├── Makefile
    │   ├── _static
    │   ├── authors.rst
    │   ├── changelog.rst
    │   ├── conf.py
    │   ├── index.rst
    │   └── license.rst
    ├── requirements.txt
    ├── setup.cfg
    ├── setup.py
    ├── src
    │   └── pypandoc
    │       ├── __init__.py
    │       └── pypandoc.py
    └── tests
        ├── conftest.py
        └── test_pypandoc.py


Note
====

This project has been set up using PyScaffold 3.1 and Cookiecutter 1.6.

For details and usage information on PyScaffold, see https://pyscaffold.org/.

For details and usage information on Cookiecutter, see https://cookiecutter.readthedocs.io/.
